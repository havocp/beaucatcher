CaseClassCodecs

class CaseClassCodecs [E <: AnyRef with Product] extends IdEncoders with ValueDecoders

Encoders/Decoders for case classes; this is expensive to create so should be cached. We don't let you use a case class for a query since the intent is unclear (just _id or all fields?).
