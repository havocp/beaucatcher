org.beaucatcher.mongo

Context

trait Context extends AnyRef

A Context represents an underlying Mongo protocol implementation, a connection pool, and a single named database on that connection. It may also include other configuration settings and the Akka ActorSystem.

Implementations of Context are provided by a "driver" such as org.beaucatcher.jdriver.JavaDriverContext.

This object is normally passed around as an implicit.
