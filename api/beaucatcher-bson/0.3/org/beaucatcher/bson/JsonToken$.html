  31. def tokenize (input: Iterator[Char]): Iterator[JsonToken]

    Tokenize a series of characters into JSON tokens.

    Tokenize a series of characters into JSON tokens. Performs no syntax or semantic checking of relationship between tokens; only validates "within" tokens, such as checking that numbers are valid numbers and strings are valid strings. But doesn't care whether numbers are next to each other or not, it just gives you a raw jumble of tokens. This is not necessarily useful for most purposes. Use the higher-level parsers.

    input

    iterator over characters

    returns

    iterator over JSON tokens
