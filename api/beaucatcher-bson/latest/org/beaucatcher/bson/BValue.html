org.beaucatcher.bson

BValue

trait BValue extends AnyRef

BValue is the base trait for all BSON value types. All BValue are immutable and the concrete subtypes of BValue are case classes.

See http://bsonspec.org/ for more information about BSON.

The names of the subclasses implementing BValue mostly match the names in the MongoDB shell such as ObjectId, ISODate, except for numbers (because there are separate types for BInt32, BInt64, and BDouble, while JavaScript has a single number type).

Most uses of BValue will start with the container BObject, which implements the standard scala.collection.immutable.Map trait. There's also BArray which implements scala.collection.immutable.LinearSeq.

A subset of BValue subtypes also implement the JValue trait, indicating that these types can appear in a JSON document, as well as in a BSON document.

JObject and JArray are separate types from BObject and BArray because they are guaranteed to contain only JValue elements, with no BSON-only values.

Import everything from Implicits if you want to auto-convert plain Scala types into BSON wrapper types. This lets you type nice BSON literals along these lines:

   BObject("a" -> 42, "b" -> "hello world", "c" -> new ObjectId())

Because the BValue subtypes are case classes, you can use pattern matching to unwrap them.
