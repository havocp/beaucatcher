  11. def fromJValue (jvalue: JValue, schema: org.beaucatcher.bson.ClassAnalysis[_ <: Product], flavor: Value = JsonFlavor.CLEAN): BValue

    Converts a JValue to a BValue using a case class as a schema to decide on destination types.

    Converts a JValue to a BValue using a case class as a schema to decide on destination types. For example, an object ID might be a BString in the JValue, and the string would be parsed as an object ID and converted to BObjectId. The mapping from JSON to BSON varies with the org.beaucatcher.bson.JsonFlavor.

  20. def parseJson (json: String, schema: org.beaucatcher.bson.ClassAnalysis[_ <: Product], flavor: Value = JsonFlavor.CLEAN): BValue

    Parse a JSON string, validating it against the given ClassAnalysis.

    Parse a JSON string, validating it against the given ClassAnalysis. The JSON string must contain all the fields found in the case class (unless the fields have a scala.Option type). If the fields have a BSON type (such as org.bson.types.ObjectId) then the returned BValue will contain BSON-only types such as BObjectId. The types of the case class fields are used to decide how to parse the JSON, for example if the _id field in the case class has type org.bson.types.ObjectId, then the parser knows to return a string in the JSON document as BObjectId rather than BString.

  26. def wrap (value: Any): BValue

    Wraps an object in a BValue.

    Wraps an object in a BValue. Throws java.lang.UnsupportedOperationException if wrapping the object is not possible.

    Because this is a dynamic (untypesafe) wrap, it's always better to use implicit conversions from Implicits unless you have an Any with unknown type. Typically this untypesafe method is needed for Java interoperability because you can end up with a java.lang.Object that needs wrapping, when dealing with say a Java collection.
