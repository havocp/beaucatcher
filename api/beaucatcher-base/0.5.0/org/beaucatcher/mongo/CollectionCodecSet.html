CollectionCodecSet

trait CollectionCodecSet [-QueryType, -EncodeEntityType, +DecodeEntityType, -IdType, +ValueType] extends ReadOnlyCollectionCodecSet[QueryType, DecodeEntityType, IdType, ValueType]

The set of encoders and decoders required to read and write a collection. EncodeEntityType and DecodeEntityType are separate so they can be contravariant and covariant.
